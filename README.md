### Environment and packages needed

- **Git:** installed and on `PATH`
- **GitHub CLI (`gh`):** [install](https://cli.github.com/), authenticated (`gh auth login`)
- **Cursor Agent CLI (`agent`):** [Cursor](https://cursor.com/) v2.4.7 (Universal) or later  
  - Install: `curl https://cursor.com/install -fsS | bash`  
  - If `agent` is not on `PATH`, set `AGENT_PATH` (or `CURSOR_AGENT_PATH`) to the full path of the agent binary
